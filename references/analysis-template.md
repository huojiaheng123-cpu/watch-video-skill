# 深度视频分析输出模板

当用户想要的不只是摘要，而是“像认真看过一样拆解”，使用这个模板。不要机械填空，按视频实际内容取舍，但必须保留证据和不确定性。

## 观看状态

- 状态：已播放 / 已读取本地媒体 / 被拦截 / 只能静态分析。
- 能力等级：minimal / recommended / full。
- 证据文件：媒体信息、关键帧、联系图、音频、转写路径。
- 未确认：列出没有看清、没听清、无法访问或不能验证的部分。
