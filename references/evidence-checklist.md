# 视频观看证据清单

观看视频后，尽量收集以下证据：

- 输入来源：本地文件路径或网页 URL。
- 最终 URL：短链接跳转后的真实页面。
- 播放状态：是否找到视频元素，是否能播放。
- 媒体信息：时长、宽高、视频编码、音频编码。
- 关键帧：至少覆盖开头、中段、结尾。
- 联系图：长视频建议每 5-15 秒抽一帧。
- 音频：是否存在口播、背景音乐或环境声。
- 转写：JSON、SRT、TXT 路径。
- 观看限制：登录墙、验证码、反爬、权限不足、音频不可访问等。

判断标准：

- 只有标题/页面文案：不能声称看过视频。
- 只有一张截图：只能做静态画面分析。
- 有多时间点关键帧：可以做画面时间线总结。
- 有音频转写：可以分析口播观点。
- 有关键帧 + 转写：可以做较完整的视频理解。

深度拆解标准：

- 能按时间线复述视频内容，而不是只给一句摘要。
- 能识别博主核心主张。
- 能区分事实、观点、预测和营销表达。
- 能给出可信度判断，而不是直接接受博主结论。
- 能结合用户的业务、学习和工具链给出启发。
- 能给出可执行下一步，而不是只写抽象建议。
